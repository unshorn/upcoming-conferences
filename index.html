How to Contribute

Contributions are welcome! The database is maintained as plain JSON files in a GitHub repository. The easiest way to contribute is to edit conferences.json or deadlines.json directly on GitHub and open a pull request.

Adding or correcting a conference deadline

  1. Go to deadlines.json on GitHub and click the pencil icon (Edit this file).
  2. Add a new JSON object to the array (or correct an existing one). Each entry follows this schema: 
    {
  "name": "49th International Conference on Software Engineering",
  "acronym": "ICSE",
  "acronymx": "ICSE'27",
  "url": "https://conf.researchr.org/home/icse-2027",
  "track": "research",
  "submission-deadline": "2026-06-30",
  "estimated": true,
  "notification": "2026-10-10",
  "cfp": "https://...",
  "venue": {
    "city": "Dublin",
    "address": "Convention Centre Dublin, Dublin, Ireland",
    "lat": 53.3498,
    "lon": -6.2603,
    "region": "Europe"
  },
  "dates": "April 25 - May 1, 2027"
}
  3. Set "estimated": true if the deadline is inferred from prior years rather than confirmed from the official CFP. Omit the field (or set it to false) for confirmed dates. Estimated dates are shown with a ~ prefix on the site.
  4. Valid values for region: Europe, Asia, Middle East, Australasia, North America, other.
  5. Only include URLs you have verified load successfully (no 404s, no placeholder domains). Use null for any URL that is not yet available.
  6. Click Propose changes, then Create pull request.

Adding a new conference series

  1. Go to conferences.json on GitHub and click the pencil icon.
  2. Add a new JSON object: 
    {
  "name": "Full conference name",
  "acronym": "ACRONYM",
  "url": "https://official-series-url.org/",
  "see-also": [],
  "core-rank": {
    "rank": "A*",
    "system": "CORE2023",
    "url": "https://portal.core.edu.au/conf-ranks/..."
  },
  "field": "software engineering",
  "sponsor": "IEEE/ACM"
}
  3. Valid values for field: software engineering, programming languages, security.
  4. Valid values for core-rank.rank: A*, A, B, C. Look up the ranking at portal.core.edu.au.
  5. Then add the corresponding deadline entries to deadlines.json as described above.
  6. Open a pull request with both files changed.

Other ways to help

  • Confirm or correct an estimated deadline (marked ~) by finding the official CFP and updating the entry.
  • Remove entries for conferences that have already taken place.
  • Fix a broken URL — set it to null or provide the correct address.
  • Open an issue to report missing or incorrect data.
